ABOUT BIXOLON CUPS DRIVER V1.3.5.x

About This Package:

                  The 'BIXOLON CUPS DRIVER V1.3.5.x printer driver was developed to add printing support for BIXOLON Thermal printer under Linux operating systems, using the widely available and popular CUPS (Common UNIX Printing System) printing system. The driver was written using the C programming language, and utilizes various CUPS API (application programming interface) calls. This driver will enable users of BIXOLON printers to print their receipt under the Linux operating systems.

Supported Operating System:

                  All Linux Distribution whose kernel version after 2.6.32-73

Assumption:

                  This BIXOLON CUPS DRIVER V1.3.5.x Package assuming that the following packages are already installed in your system:

                        1. CUPS server & architecture (see www.cups.org)
                        2. cups and cups-devel packages
                        3. ghostscript library packages
