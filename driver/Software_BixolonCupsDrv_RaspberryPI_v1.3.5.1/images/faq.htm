1.How many files are in BIXOLON CUPS Driver V1.3.5.x Installation CD and what are they?

The BIXOLON CUPS Driver V1.3.5.x Installation CD contains 6 files and 2 folder that are
            rastertoBixolon_v1.3.5.x, rastertoBixolon_v1.3.5.x.o, SoftwareLicense.txt, setup_v1.3.5.sh, readme.htm, uninstaller_v1.3.5.sh, and two folders called images and Bixolon



2. Could you please expalin about the files in the BIXOLON CUPS Driver V1.3.5.x Installation CD?

rastertoBixolon_v1.3.5.x
            CUPS Driver for BIXOLON Printer
setup_v1.3.5.sh
            Is a shell script file used to install the BIXOLON CUPS Driver V1.3.5.x in console mode
readme.htm
            Is a html based user manual for BIXOLON CUPS Driver V1.3.5.x
uninstaller_v1.3.5.sh
            Is a shell script file used to uninstall the BIXOLON CUPS Driver in console mode
images
            This folder contains some images and html files, which are useful for readme.htm file
Bixolon
            Postscript Printer Description file for BIXOLON Printer



3. I'm Very new to CUPS and is the first time i'm going to install BIXOLON CUPS Driver. What will i do first?

If you are very new to CUPS. Kindly go to http://127.0.0.1:631/help/ This will give you overall information about CUPS
if you are installing BIXOLON CUPS Driver please refer readme.htm carefully before begins installation.



4. Will this BIXOLON CUPS Driver V1.3.5.x supports all Linux Distributions?

This BIXOLON CUPS Driver V1.3.5.x will Supports All Linux Distributions whose Kernel version is after 2.6.32-73



5.My Linux Kernel version is older than the 2.6.32-73 what will I do now?

If your Kernel version is older than 2.6.32-73 . Please Contact BIXOLON CO Ltd, with your Linux Kernel version information and your Linux Distribution information. We will send the new driver to you as soon as we received your request.



6.How do I check whether BIXOLON CUPS Driver V1.3.5.x successfully installed in my Linux Distribution?

          Run the following Commands
                    ls /usr/share/cups/model/Bixolon
                    ls /usr/bin/rastertoBixolon_v1.3.5.x
                    ls /usr/lib/cups/filter/rastertoBixolon_v1.3.5.x
Successful returns of all 3 commands will indicate successful installation of BIXOLON CUPS Driver V1.3.5.x



7.BIXOLON CUPS Driver printer setting through Open Office is not taking effect. What could be the reason?

This problem is not related with our driver, is related with the linkage between the Open Office and CUPS. Since our driver is based on CUPS, whatever the setting you made through Open Office will override by CUPS admin. So please use CUPS Admin tool to change the printer settings.
