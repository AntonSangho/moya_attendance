Welcome to CUPS Admin Wizard

This Wizard will help you to Add or Modify or Configure BIXOLON Printers. Before Using CUPS Admin Wizard make sure that you have alrady installed BIXOLON CUPS Driver V1.3.5.x, if not, install it and then perform the following steps to Add or Modify or Configure BIXOLON Printer.

Setp1 : Goto http://localhost:631/admin or http://127.0.0.1:631/admin using your favorite web browser this will shows the following screen




Step2 : Click Add Printer Button to add a new printer that will shows the following screen




Step3 :The Above Screen will shows the list of all available devices (Parallel     Serial     Socket etc ) and usb printer. In that you have to select the one you want to use then click the Continue Button this will shows the following screen




Step4 : In the above screen give the printer name     Location and Description then click Continue button this will shows the following screen



Step5: The Above Screen will shows the list of all available Printer Manufacturer Information. In that you have to select Bixolon then click the Continue Button this will shows the following screen



Step6: The Above Screen will shows the list of all available Bixolon Printer Drivers installed in your system. In that you have to select the right driver for your printer then click the Add Printer Button this will shows the following screen



Step7: In the Above Screen you can modify the printer options and then click the Set Default Options Button this will apply the changes to the printer and successfully install your printer and will shows the following screen.
